# 🤖 BTC/USDT 봇 통합 관리

이 폴더는 여러 개의 트레이딩 봇을 통합 관리하고 웹 대시보드로 모니터링하기 위한 프로젝트입니다.

## 🚀 봇 실행 방법 (터미널)

### 1. 봇 시작 (Start)

대시보드와 모든 봇을 백그라운드에서 시작합니다.

```bash
./start_dashboard.sh
```

### 2. 봇 종료 (Stop)

실행 중인 모든 봇 프로세스를 종료합니다.

```bash
./stop.sh
```

### 3. 로그 확인 (Logs)

실시간 로그를 확인합니다. (종료하려면 `Ctrl + C`)

```bash
./logs.sh
```

### 4. 재시작 (Restart)

봇을 종료했다가 다시 시작합니다.

```bash
./restart.sh
```

## 🌐 웹 대시보드 접속

봇이 실행 중일 때 웹브라우저에서 아래 주소로 접속하세요.

- **로컬 접속:** [http://localhost:8000](http://localhost:8000)
- **외부/모바일 접속:** `http://<이 컴퓨터의 IP>:8000`
