# 🤖 BTC/USDT 트레이딩 봇 통합 패키지 (Server Edition)

이 패키지는 **Ubuntu 서버**에 최적화된 BTC 자동매매 봇 통합 솔루션입니다.
**설치부터 실행, 로그 관리까지 원클릭**으로 해결되도록 구성되었습니다.

---

## ✨ 주요 기능

1. **다중 전략 통합**: 5분봉, 15분봉, 30분봉, 1시간봉 봇을 동시에 구동 & 중앙 관리.
2. **원클릭 설치**: `install_system.sh` 하나로 파이썬, Docker, SSH, 절전 방지 등 모든 환경 자동 세팅.
3. **웹 대시보드**: PC/모바일 어디서든 실시간 수익률 및 차트 확인 가능.
4. **로그 자동 관리**: 매일 자정 로그 압축 & 7일 지난 로그 자동 삭제 (용량 걱정 NO).
5. **자동 재시작**: 서버가 재부팅되어도 봇이 자동으로 다시 켜집니다.

---

## �️ 처음 설치 방법 (새 서버)

USB를 꽂고 폴더를 바탕화면 등에 복사한 뒤, 터미널을 열고 아래 명령어를 입력하세요.

### 1. 설치 스크립트 실행

이 명령어 한 번이면 **모든 준비가 끝납니다.** (약 5~10분 소요)

```bash
cd btc_usdt_bot_usb_copy
chmod +x install_system.sh
./install_system.sh
```

> **설치되는 것들:**
>
> * 시스템 업데이트 & 필수 도구 설치
> * Python 가상환경 및 라이브러리 설치
> * Docker 및 Docker Compose 설치
> * **SSH 서버 (원격 접속용)**
> * **절전 모드 방지 설정 (24시간 가동)**
> * 시스템 재부팅 시 **봇 자동 실행 등록**
> * 로그 **자동 삭제(Rotation) 스케줄러 등록**

설치가 끝나면 **자동으로 대시보드가 실행**됩니다.

---

## 🚀 봇 관리 명령어

설치 후에는 아래 명령어들로 봇을 쉽게 관리할 수 있습니다.

### ▶️ 봇 시작 (Start)

```bash
./start_dashboard.sh
# 또는 간단히
./start.sh
```

### ⏹️ 봇 종료 (Stop)

```bash
./stop.sh
```

### 🔄 재시작 (Restart)

```bash
./restart.sh
```

### 📜 로그 확인 (Logs)

실시간 로그를 봅니다. (빠져나오려면 `Ctrl + C`)

```bash
./logs.sh
```

---

## 🌐 모니터링 (웹 대시보드)

같은 와이파이(공유기)를 쓰는 핸드폰이나 PC에서 접속 가능합니다.

1. **접속 주소**: `http://<서버_IP_주소>:8000`
    * 예: `http://192.168.0.15:8000`
2. **기능**:
    * 실시간 수익률 확인
    * 봇 개별 끄기/켜기
    * 트레이딩 차트 조회 (손절/익절 라인 표시)

---

## 📂 폴더 구조 설명

* `install_system.sh`: **통합 설치 스크립트 (가장 중요)**
* `start_dashboard.sh`: 봇 실행 스크립트
* `bot_manager.py`: 봇들을 관리하는 메인 관제탑
* `setup_log_rotate.sh`: 로그 관리 스케줄러 (설치 시 자동 실행됨)
* `bybit_bot_usb(...)`: 각 전략별 봇 소스코드 폴더들

---

**⚠️ 주의사항**

* 이 폴더를 다른 곳으로 옮기면 `install_system.sh`를 다시 한 번 실행해 주는 것이 좋습니다. (자동 실행 경로 갱신을 위해)
