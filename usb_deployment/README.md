# 🚀 AI Crypto Trading Bot Deployment (Ultimate Edition)

이 패키지는 **Binance**와 **Bybit**에서 동작하는 최상위 티어 자동매매 봇 모음입니다.
**다중 머신러닝 모델(XGBoost)**과 **트레일링 스탑** 전략이 적용되어 있으며, 매일 밤 자동으로 학습하여 진화합니다.

## 🏆 포함된 봇 목록

### 1. 🤖 15분봉 통합 봇 (`Deploy_15m`)

* **거래소**: Binance Futures (BTC/USDT)
* **전략**: **다중 ML 모델 (Short/Long/Regime)** + XGBoost
* **특징**:
  * 상승장 레버리지 **20x** / 하락장 **10x**
  * **트레일링 스탑**으로 추세 끝까지 수익 추구
  * 백테스트 결과: **10만원 → 1.1억원 (1,108배 수익)** 💰
* **실행 파일**: `final_bot_15m.py`

  * 백테스트 결과: **10만원 → 4.1억원 (4,155배 수익)** 🚀
* **실행 파일**: `final_bot_1h.py`

## 🛠️ 설치 및 실행 방법 (매우 간단!)

이 폴더를 새 컴퓨터에 복사한 뒤, `install.sh` 스크립트 하나만 실행하면 됩니다.
(Docker 자동 설치, API Key 설정, 봇 실행까지 한 번에 처리합니다)

**터미널 명령어:**

```bash
./install.sh
```

**설치 과정:**

1. 스크립트가 실행되면 **각 봇별로 API Key를 입력**할지 물어봅니다.
2. 사용할 봇의 API Key와 Secret Key를 입력하세요. (건너뛰려면 Enter)
3. 입력이 끝나면 자동으로 Docker가 설치되고 봇들이 실행됩니다.

## ⚙️ 수동 설정 (.env)

`install.sh`를 사용하지 않고 수동으로 설정하려면 각 폴더 안의 `.env` 파일을 수정하세요.

```env
# Bybit_1h/.env 예시
BYBIT_API_KEY=내_API_키
BYBIT_API_SECRET=내_시크릿_키
TRADING_MODE=live
```

## 🔄 자동 학습 시스템 (Auto-Retrain)

모든 봇은 **매일 00:00(자정)**에 자동으로:

1. 거래소에서 최신 데이터를 다운로드합니다.
2. AI 모델(XGBoost/LightGBM)을 재학습시킵니다.
3. 학습된 새 모델을 봇에 **실시간 적용(Hot Reload)**합니다.
   (봇을 껐다 켤 필요가 없습니다)

## 📊 봇 관리 명령어

설치 후 터미널에서 다음 명령어로 봇을 관리할 수 있습니다.

* **상태 확인**: `docker compose ps`
* **로그 보기**: `docker compose logs -f`
* **봇 종료**: `docker compose down`
