# BTC 30M Adaptive Extreme Growth Strategy (Live Ready)

본 시스템은 비트코인(BTC/USDT) 30분봉 데이터를 기반으로 하는 **적응형(Adaptive)** 고수익 추세 추종 전략입니다. 2017년부터 2025년까지의 모든 시장 국면에서 검증되었으며, 특히 2025년과 같은 저변동성 구간에서도 휩소를 방지하도록 설계되었습니다.

## 📂 파일 구조
- `strategy.py`: 전략의 핵심 로직이 담긴 클래스 파일입니다. (진입/청산, 적응형 필터, 리스크 관리 포함)
- `live_bot.py`: 바이낸스 선물 API와 연동되어 실시간으로 매매를 수행하는 봇입니다.
- `run_backtest.py`: 전 기간 연속 복리 백테스팅을 실행합니다.
- `verify_years.py`: 연도별 독립적인 성과를 검증합니다.
- `requirements.txt`: 봇 구동에 필요한 라이브러리 목록입니다.
- `.env.example`: API 키 설정을 위한 환경 변수 템플릿입니다.

## 🚀 전략 핵심 요약
1. **모드**: `extreme_growth` (적응형 익스트림 성장 모드)
2. **타임프레임**: 30분 (30m)
3. **주요 특징 (Adaptive Logic)**:
   - **저변동성 필터**: ATR %가 0.35% 미만인 구간에서는 더 엄격한 진입 조건을 적용하여 횡보장 휩소 방지.
   - **강세장 홀딩 (Strong Bull)**: 강력한 상승장에서는 `bull_exit_slow`(48시간 저점)를 사용하여 추세를 끝까지 추적.
   - **실시간 신호 동기화**: `live_bot.py`는 `strategy.py`의 `get_current_signal` 함수를 통해 백테스트와 100% 동일한 로직으로 동작.

4. **리스크 관리**:
   - **가변 레버리지**: 시장 상황 및 신뢰도에 따라 5x ~ 최대 50x까지 동적 조절.
   - **리스크 기반 수량**: 매 거래 시 잔고의 **9%** 리스크를 할당하여 복리 효과 극대화.
   - **3단계 분할 익절**:
     - **Long**: 레버리지 ROI 50%, 120%, 300% 지점 단계적 익절.
     - **Short**: 레버리지 ROI 40%, 100%, 250% 지점 단계적 익절.
   - **적응형 트레일링 스탑**: 수익권 진입 시 Ratchet 로직을 통해 빠르게 익절가 보호.

## 🤖 실거래 자동매매 (Live Trading)
### 1. 환경 설정
- **Python 3.8+** 필요
- **의존성 설치**: `pip install -r requirements.txt`
- **API 키 설정**: `.env.example`을 `.env`로 복사 후 바이낸스 API Key/Secret 입력.

### 2. 실행 및 운영
- `python live_bot.py` 명령어로 실행합니다.
- 봇은 1분마다 시장 데이터를 체크하며, 새로운 봉이 생성될 때 또는 수익권/손절권 진입 시 즉각 대응합니다.
- **포지션 동기화**: 봇 시작 시 현재 거래소에 잡힌 포지션을 자동으로 인식하여 연동합니다.

## ⚠️ 주의사항
- 본 전략은 고레버리지를 사용하므로, 거래소의 증거금 비율 및 최대 포지션 제한을 반드시 숙지해야 합니다.
- 실거래 전 반드시 소액으로 로직 동작(분할 익절, 트레일링 스탑 등)을 확인하시기 바랍니다.
